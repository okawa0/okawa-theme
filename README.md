# Okawa Portfolio

和モダン × ミニマル をテーマにしたポートフォリオサイト。

![screenshot](./screenshot.png)

## Demo
https://okawa-web.com

## Overview
フロントエンドのスキルをまとめた個人ポートフォリオ。  
アニメーション・レスポンシブ対応・WordPressテーマ開発などの実力を確認できる構成。

## Tech Stack
- HTML / SCSS
- JavaScript
- WordPress (カスタムテーマ)
- GitHub

## Features
- フェードアップアニメーション
- ローディング演出
- レスポンシブデザイン
- 作品一覧のカード UI
